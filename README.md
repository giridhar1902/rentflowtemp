# PropTech Manager Monorepo

## Apps

- `apps/mobile-web`: React + Vite + Capacitor-ready client.
- `apps/api`: NestJS API (foundation scaffold).

## Tooling

- Package manager: `pnpm`
- Task runner: `turbo`

## Common Commands

```bash
pnpm install
pnpm dev
pnpm lint
pnpm typecheck
pnpm test
pnpm build
```

## Environment

Copy `.env.example` and each app-level `.env.example` file into local `.env` files before running.
